Create Customisable Pre-School Tracing Resources!

Welcome to Tracey, a tool for creating printer-friendly customisable tracing resources which can be used for activities with pre-school children in nurseries and at home. Use the form below to get started.

Printer friendly! When you print this page, we won't print out everything, just your personalised resources. Each child's sheet will start on a separate page.

Tracey does not support mobile versions of the Firefox web browser - but the desktop version works just fine!

Instructions

Child Name (required): Fill in the child's name to personalise their printed worksheet. For the Names form, this is also what will be printed out for the child to trace.

Font Size (required): Choose between large, medium or small font size (font heights of 4cm, 3cm and 2cm respectively when printed).

Repeats (required): How many times you want each name, word/sentence, alphabet or set of numbers to be printed out for the particular child's worksheet.

Words (required on Words form only): The desired word or sentence (1-40 characters) to be printed out for the child to trace.

Case (required on Alphabet form only): Specify whether you want the alphabet to be printed in either upper or lower case for the child to trace.

Numbers (required on Numbers form only): Specify whether you want to print out the numbers 1-10 or 1-20 for the child to trace.
